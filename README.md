Pipeline for assessing and rewriting emails using a LLM.

Contains 5 Python files:
- PipelineLoopNoGen.py = File that should be executed. Loads all necessary files and calls for the feedback loop.
- FeedbackLoop.py = Defines the feedback loop, including email assessment and email rewriting.
- EmailAssessmentPrompts.py = Includes the prompts for the LLM to assess the email on multiple dimensions.
- FeedbackLoopPrompts.py = Includes the prompts for the LLM to write instructions and rewrite the email.
- utils.py = Includes the code for device selection and parsers.

Other files included:
- Files/Knowledge Base/kb.json = Knowledge base in json format. Previously extracted from an image using a LLM. Includes information on what persuasion techniques are effective for specific types of people.
- Files/Profiles/1EleanorVance.json and 4DanielHayes.json = User profiles previously generated by a LLM of fictional people.
- Files/example.eml and example2.eml = Example emails that can be rewritten using the loop.
